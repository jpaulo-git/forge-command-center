# Forge
A personal productivity command center designed to combine task management, habit tracking, goal planning, and personal reflection into one customizable application.

## Overview
Forge is starting as a Python CLI application while I learn software development fundamentals. The long-term goal is to evolve it into a full-stack productivity dashboard.

## Why I Built This
I wanted to create a productivity system that I would use every day while building my programming skills.

Forge is designed to grow alongside my development journey by adding new features, improving architecture, and eventually becoming a full web application.

## Current Features

### Task Management
- Create tasks
- View tasks
- Complete tasks
- Delete tasks
- Persist data between sessions using JSON storage

## Planned Features
- Habit Tracking
- Goal Management
- Daily Journal
- Weekly Reviews
- Reports
